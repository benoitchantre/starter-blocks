A simple WordPress plugin boilerplate focused on custom block development.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/starter-block-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Example blocks will be available in the block editor.
4. Start creating your own blocks in the `src/` directory.
